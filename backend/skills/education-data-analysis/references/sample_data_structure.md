# 教育学情数据结构说明

## 数据格式要求

### Excel文件格式 (.xlsx, .xls)

支持多级表头的Excel文件。默认跳过前3行表头，从第4行开始读取数据。

#### 表头结构示例

第1行：办学规模 | 办学性质 | 学校代码 | 年级 | 班级 | 考号 | 姓名 | 性别 | 成长环境 | 成长环境 | 成长环境 | 成长环境 | 学生发展 | 学生发展 | 学生发展 | 学生发展 | 学生发展 | 学生发展

第2行：办学规模 | 办学性质 | 学校代码 | 年级 | 班级 | 考号 | 姓名 | 性别 | 亲子关系 | 师生关系 | 同伴关系 | 校园安全 | 身心健康 | 情绪状态 | 运动健康 | 学习创新机会 | 学习习惯 | 学业达标

第3行：办学规模 | 办学性质 | 学校代码 | 年级 | 班级 | 考号 | 姓名 | 性别 | 亲子关系 | 师生关系 | 同伴关系 | 校园安全 | 身心健康 | 情绪状态 | 运动健康 | 学习创新机会 | 学习习惯 | 学业达标

第4行开始：实际数据行

#### 数据行示例

大规模 | 公办 | 19 | 七年级 | 10班 | 202601001 | 学生001 | 男 | 75 | 100 | 66.75 | 75 | 87.5 | 50 | 91.75 | 100 | 100 | 100

### CSV文件格式 (.csv)

标准CSV格式，包含完整的列名。

#### 列名要求

```
办学规模,办学性质,学校代码,年级,班级,考号,姓名,性别,成长环境_亲子关系,成长环境_师生关系,成长环境_同伴关系,成长环境_校园安全,学生发展_身心健康,学生发展_情绪状态,学生发展_运动健康,学生发展_学习创新机会,学生发展_学习习惯,学生发展_学业达标
```

#### 数据行示例

```csv
大规模,公办,19,七年级,10班,202601001,学生001,男,75,100,66.75,75,87.5,50,91.75,100,100,100
```

## 必需列

### 基本信息列

| 列名 | 类型 | 说明 |
|------|------|------|
| 办学规模 | 字符串 | 微规模、小规模、中规模、大规模 |
| 办学性质 | 字符串 | 公办、民办 |
| 学校代码 | 字符串/数字 | 学校唯一标识 |
| 年级 | 字符串 | 学生年级 |
| 班级 | 字符串 | 学生班级 |

| 性别 | 字符串 | 男、女 |

## 数据质量要求

### 必需要求

1. **完整性**：所有必需列必须存在
2. **数值范围**：指标得分应在0-100范围内
3. **数据类型**：指标列必须为数值类型
4. **性别值**：只能为"男"或"女"
5. **办学性质**：只能为"公办"或"民办"
6. **办学规模**：只能为"微规模"、"小规模"、"中规模"、"大规模"

### 推荐要求

1. **样本量**：建议至少100名学生
2. **学校数**：建议至少3所学校
3. **完整性**：无缺失值（缺失值会被自动删除）
4. **脱敏处理**：姓名和考号应已脱敏处理

## 数据处理流程

### 自动处理步骤

1. **表头跳过**：自动跳过前3行表头（Excel文件）
2. **类型转换**：将指标列转换为数值类型
3. **脏数据清理**：识别并清理非数据行
4. **缺失值处理**：删除含缺失值的行
5. **综合得分计算**：
   - 成长环境均分 = 4个成长环境指标平均值
   - 学生发展均分 = 6个学生发展指标平均值

### 四象限划分

- **划分标准**：基于成长环境和学生发展的中位数
- **第一象限**：成长环境 ≥ 中位数 且 学生发展 ≥ 中位数
- **第二象限**：成长环境 < 中位数 且 学生发展 ≥ 中位数
- **第三象限**：成长环境 ≥ 中位数 且 学生发展 < 中位数
- **第四象限**：成长环境 < 中位数 且 学生发展 < 中位数

```json
{
  "indicators": {
    "environment": ["亲子关系", "师生关系", "同伴关系", "校园安全"],
    "development": ["身心健康", "情绪状态", "运动健康", "学习创新机会", "学习习惯", "学业达标"]
  },
  "indicator_names": {
    "亲子关系": "亲子关系",
    "师生关系": "师生关系",
    ...
  }
}
```
