# 数据规范文档

## 概述

本文档定义了 `dashboard-generator` skill 所需的数据文件格式和规范。

## 数据文件列表

### 必需文件（6个）

1. **indicator_stats.csv** - 各指标统计数据
2. **gender_analysis.csv** - 性别差异分析数据
3. **scale_analysis.csv** - 办学规模分析数据
4. **nature_analysis.csv** - 办学性质分析数据
5. **quadrant_analysis.csv** - 四象限分析数据
6. **school_analysis.csv** 或 **school_position.csv** - 学校层面数据

### 可选文件（3个）

1. **intervention_suggestions.csv** - 干预建议数据
2. **analysis_summary.json** - 分析摘要数据
3. **analysis_data_clean.csv** 或 **analysis_data.csv** - 基础数据

---

## 详细规范

### 1. indicator_stats.csv

**用途**：各指标统计数据，用于生成平均分对比图和风险暴露率图

**必需列**：

| 列名 | 数据类型 | 说明 | 示例 |
|------|---------|------|------|
| 指标 | 字符串 | 指标名称 | "亲子关系" |
| 平均分 | 数值 | 该指标的平均得分 | 70.15 |
| 风险暴露率(%) | 数值 | 低于60分的百分比 | 30.50 |

**示例数据**：
```csv
指标,平均分,风险暴露率(%)
亲子关系,70.15,30.50
师生关系,81.98,14.11
同伴关系,56.42,51.95
校园安全,83.79,16.32
身心健康,57.28,46.75
情绪状态,76.56,15.85
运动健康,85.24,9.93
学习创新机会,61.36,54.04
学习习惯,64.04,44.05
学业达标,90.08,9.92
```

### 5. quadrant_analysis.csv

**用途**：四象限分析数据，用于生成四象限分布图

**必需列**：

| 列名 | 数据类型 | 说明 |
|------|---------|------|
| 象限 | 字符串 | 四象限名称 |
| 成长环境_均分 | 数值 | 该象限成长环境平均分 |
| 学生发展_均分 | 数值 | 该象限学生发展平均分 |
| 人数 | 整数 | 该象限学生人数 |

**象限名称规范**：
- 第一象限-优势发展型
- 第二象限-逆境成长型
- 第三象限-潜力待发型
- 第四象限-重点关注型

**示例数据**：
```csv
象限,成长环境_均分,学生发展_均分,人数
第一象限-优势发展型,85.43,82.9,3481
第二象限-逆境成长型,66.72,77.85,1187
第三象限-潜力待发型,81.07,65.75,1189
第四象限-重点关注型,60.13,62.34,3467
```

### 9. analysis_data_clean.csv 或 analysis_data.csv（可选）

**用途**：基础数据文件，用于获取准确的统计信息（总人数、性别分布等）

**必需列**：
- 性别
- 学校代码
- 其他指标列

**说明**：如果此文件存在，仪表盘将使用此文件计算准确的统计数据。

## 数据格式要求

### 文件格式
- **编码**：UTF-8 或 UTF-8-SIG（带BOM）
- **分隔符**：逗号（,）
- **行结束符**：LF 或 CRLF
- **表头**：第一行为列名

### 数值格式
- **整数**：不包含小数点（如：3481）
- **小数**：包含小数点（如：70.15）
- **百分比**：数值形式，不带%符号（如：30.50）

### 字符串格式
- **引号**：可选，包含逗号的字符串必须用引号
- **编码**：UTF-8

## 数据验证

### 必需文件检查
生成仪表盘前，系统会检查以下文件是否存在：
- ✅ indicator_stats.csv
- ✅ gender_analysis.csv
- ✅ scale_analysis.csv
- ✅ nature_analysis.csv
- ✅ quadrant_analysis.csv
- ✅ school_analysis.csv 或 school_position.csv

### 数据格式检查
系统会验证：
- ✅ 列名是否正确
- ✅ 数据类型是否匹配
- ✅ 数值范围是否合理（如：得分在0-100之间）

---

## 数据生成

### 从education-data-analysis生成

运行 `education-data-analysis` skill 会自动生成所有必需的数据文件：

```bash
python education_analysis.py --data 学生指标得分清单.xlsx --output output/
```

生成的数据文件位于 `output/data/` 目录下。

### 手动创建数据文件

如果不使用 `education-data-analysis`，可以手动创建符合规范的数据文件。

**注意事项**：
1. 确保所有必需列存在
2. 确保数据类型正确
3. 确保文件编码为UTF-8
4. 确保列名与规范完全一致

---

## 数据安全

### 敏感数据处理
- ⚠️ 学生姓名应已脱敏
- ⚠️ 考号等个人信息应已脱敏
- ⚠️ 学校代码可使用编号替代真实校名

### 数据传输
- 建议使用加密传输
- 不在公网暴露数据文件
- 定期清理临时数据文件
